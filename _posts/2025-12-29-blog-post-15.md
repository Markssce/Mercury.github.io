## 前置工作：

1. 配置python环境；
2. 下载miniconda；
3. 下载gcc；

```
brew install gcc
```

4. 下载gprMAX；

```
conda update conda
conda install git
git clone https://github.com/gprMax/gprMax.git
```

---

## 具体安装步骤

1. 创建并激活独立虚拟环境（不要放在项目目录内）

MacOS:(Win)
```
# 先搞一个虚拟环境并启用
cd ~/Documents
python3 -m venv gprmax-env
source gprmax-env/bin/activate
```

2. 升级工具链并安装构建依赖（必须提前装！）

````
# 下载需要的库
python -m pip install --upgrade pip setuptools wheel
python -m pip install numpy Cython
````

3. 进入 gprMax 源码目录，使用 --no-build-isolation 安装

```
cd gprMAX-1/gprMax-master
python -m pip install -e . --no-build-isolation
```

4. 验证是否下载成功：

```
python -c "import gprMax; print('✅ gprMax version:', gprMax.__version__)"

```
